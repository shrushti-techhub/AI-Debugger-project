# AI Debugger Pro 🤖🐞

AI Debugger Pro is a smart tool designed to help developers detect and fix bugs in their code quickly and efficiently using AI-powered analysis.

## 🚀 Features

* 🔍 Detect common coding errors automatically
* ⚡ Suggest fixes for bugs in real-time
* 🧠 AI-powered code analysis
* 💻 Supports multiple programming languages
* 📊 Improves code quality and debugging speed

## 🛠️ Technologies Used

* Python
* Machine Learning / AI
* Git & GitHub
* VS Code

## 📂 Project Structure

```
ai-debugger-pro
│
├── main.py
├── debugger.py
├── requirements.txt
└── README.md
```

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/your-username/ai-debugger-pro.git
```

2. Navigate to the project directory

```
cd ai-debugger-pro
```

3. Install dependencies

```
pip install -r requirements.txt
```

4. Run the program

```
python main.py
```
